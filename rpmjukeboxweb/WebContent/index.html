RPM Jukebox

Long ago in a galaxy far, far away there existed a Flash based Jukebox hosting all the songs submitted to the RPM Challenge up until that point. It made it possible to take a random journey through the body of collected work and store any favourites in a playlist. Unfortunately it was buggy and the original maintainer stopped work on it, so it was pulled and we are left with the browser based version we have today - which while functional, does not provide the same means to discover the great music hidden beneath.

So with that in mind, I've taken it upon myself to restore the same kind of easy access in a new RPM Jukebox...

.deb (Ubuntu, Mint etc) .rpm (Fedora etc) .tar.gz (Manual install)

Functionality includes:-

  • Search over all currently available tracks hosted on the RPM Challenge site
  • The ability to mark a track as a favourite and for it to appear in the favourites playlist
  • Create custom playlists and import/export them for sharing
  • Play a random selection of songs from the RPM catalogue
  • Display of album/artist images where available (I think most images prior to 2016 have been lost to the ether)
  • File caching to take the load off the RPM servers
  • Shuffle/repeat play
  • Graphic equalizer

Source code available on GitHub - pull requests welcome. Please report any issues you discover on the GitHub issues page.

Notes

On Windows you may see a firewall warning pop-up like the screenshot below. This is due to the way the file caching works and you can safely leave the jukebox blocked by hitting cancel, it doesn't need the external access, it's just Windows being dumb.
